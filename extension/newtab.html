Grok Terminal

A real local shell (zsh) living as a first-class browser tab.

Start the terminal server once:
bun run terminal

Then open http://localhost:4321

In Helium you can now hit Cmd+T and get a real terminal instead of a new-tab page.
Load this extension folder in chrome://extensions (dev mode) and it will redirect here automatically.

This is the minimal Level-1 MVP of “terminal as a browser tab” for the grok-mission-control project.
The richer agent-orchestration dashboard lives on the main branch.
